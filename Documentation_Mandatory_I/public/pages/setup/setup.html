Guide to Your NodeJs Project set-up

entrypoint -> package.json

type: module(ALT er moduler. gør muligtat benytte sig af import og export - kan man ikke i common js, Importering af module gør at man ikke har adgang til filer som i node hvorfor man importerer path til at give stivejen), express, scripts

app.js -> deklaration, listen og routes(benytter sig af callback functions), tolk data som json (fortæller vi benytter os af bodyparser, json fordi alle sprog kan parse det) og gør public mappe tilgængelig (klient får adgang til mappe med statiske filer)

template engine

ssr af sider -> nodemon.json i rod -> så nodemon genkøres altså modtager det aktuelle indhold på siderne, readFileSync

Forskellige måder at sende: send({ data OR message: xxx }), sendFile(__dirname + "fil.html" and use of path module)

Tilføjelse af miljø variabler -> fjerne ansvar fra koden
