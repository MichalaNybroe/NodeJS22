Commands and tools

Package manager

npm

Npm is the build tool for NodeJs. Therefore tools and the setup of the package.json is installed through npm commands.

Display version of package manager

npm -v

Install dependencies in a node_modules folder

npm install

Install "Express"

Express is a web application framework often used when working with NodeJs.

npm install express

Install everything in package.json

npm i

Development environment

Nodemon

Nodemon is a development environment tool used to run a server.

When nodemon is used it re-runs itself everytime changes are being made in json or js. Other file types can be added in a file nodemon.json - example below.

Install nodemon globally

npm install -g nodemon

Run server

nodemon.cmd app.js

Use of environment variables

Cross-environment variables

Write npm install --save-dev cross-env to add the "devDepencies" in package.json.


When enabling the use of environment variables while using nodemon it will not be enabled when nodemon does a re-run. Therefore you can add the variable to your development script before nodemon.cmd app.js to ensure it being enabled without having to stop and re-run the whole program.

To refer to environment variable in app.js write process.env.PORT.

Svelte

Svelte is a web framework in which one can define components. When using Svelte one can make changes in DOM without keeping an overview.

Install Svelte

npm create vite

Run your svelte project

npm run dev

Set-up

Enable the extension "Svelte for VS Code" and when a Svelte project is created the following is deleted: all content in app.svelte, folder named lib and svg file in assets.
